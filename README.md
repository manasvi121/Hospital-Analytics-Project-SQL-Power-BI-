# Hospital-Analytics-Project(SQL-Power-BI)
Project Overview

This project demonstrates an end-to-end healthcare analytics workflow, starting from raw hospital datasets, performing SQL-based analysis, and finally building an interactive Power BI dashboard.

The objective of this project is to analyze:

Doctor performance and workload

Hospital revenue trends

Patient visit patterns and segmentation

Treatment demand frequency

This project is designed to reflect industry-style data analytics practices and is suitable for internships and entry-level Data Analyst roles.

Tools & Technologies Used

MySQL – Database creation, joins, aggregations, analytics queries

SQL – Data modeling, analysis, and feature engineering

Power BI Service (Browser) – Dashboard creation and visualization

Excel / CSV – Intermediate data preprocessing (browser-friendly approach)
